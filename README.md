# サンライズ瀬戸・出雲 空席監視システム

JRサンライズ瀬戸・出雲の空席状況を自動監視し、空席が出たら音で通知するCLIアプリケーションです。

## セットアップ

```bash
bun install
```

## 使い方

```bash
bun start
```

起動すると対話式で以下の情報を入力します：

1. 列車選択（サンライズ瀬戸/出雲）
2. 乗車駅
3. 降車駅
4. 乗車日
5. 部屋タイプ

設定は保存可能で、次回起動時に再利用できます。

## 機能

- 30秒ごとに自動チェック
- 空席発見時に音声通知
- 設定の保存・再利用
- 対話式の設定入力

## 終了方法

`Ctrl + C` で終了します。
